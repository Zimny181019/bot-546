Privacy Policy

🤖 Bot: ZR Points
📅 Effective Date: July 3, 2026
This Privacy Policy explains what data the bot processes on Discord servers, why it is processed, and how long it is retained.

📋 Data We Collect

  • 👤 User IDs
  • 🏠 Server (Guild) IDs
  • 💬 Channel IDs where activity takes place
  • 🎭 Role IDs (used for permission checks and temporary role assignment)
  • 📝 Selected characteristics of message content (see section below)
  • 🔊 Voice channel activity (join/leave timestamps, mute/deafen state)
  • 😀 Reactions (emoji used, ID of the message reacted to)
  • 💰 Virtual point balance and transaction history
  • 🎮 Game statistics (wins/losses, amounts wagered)
The bot does not use the Presence Intent (online/offline status of other users) — this is not required by, and is not used by, any bot feature.

Message Content — What Exactly Is Analyzed

The bot requires the Message Content Intent because some of its features (task system, ad-link detection, anti-spam, the boss-fight event) need to analyze message content in real time. The bot only extracts the characteristics listed below — it does not permanently store the full text of messages:

  • 🔗 Presence of an invite link (discord.gg) — used to detect server advertisements in designated channels
  • 🖼️ Presence of an image, GIF, or sticker — used for point rewards and daily tasks
  • ❓ Presence of a question mark / emoji in the text — used for themed tasks
  • 🔁 Comparison with the user's previous message — anti-spam mechanism (duplicate/copy-paste detection)
  • ⚔️ The literal word "attack" in a designated channel — a text command used in the boss-fight event
Extracted data (e.g. "number of messages containing an image") is stored as aggregated counters, not as the original message text.

🗄️ Data Storage

Data is stored locally on the bot's hosting server and used exclusively to support the features described in this Policy. Wallet and statistics data are periodically reset as part of the recurring points-system reset cycle (every 60 days by default) — resets are announced on the server in advance.

🤝 Data Sharing

  • ✗ Data is never sold or shared with third parties
  • ✗ Data is never used for advertising purposes outside the server
  • ✓ Data may be visible to server administrators as part of standard moderation features (e.g. activity logs, transaction history)

👶 Children

The bot is intended to be used in accordance with Discord's Terms of Service, which require users to be at least 13 years old (or the minimum age required in their country). We do not knowingly collect data from users below this age.

✋ Your Rights

You may request deletion of your data (wallet, statistics, history) at any time by contacting server staff through the ticket system. You may also stop using the bot at any time — your data will no longer be actively collected.

✏️ Changes to This Policy

We may update this Privacy Policy from time to time. Significant changes will be announced on the server. Continued use of the bot after changes take effect constitutes acceptance of those changes.

✉️ Contact

For any questions regarding privacy or data handling, please contact server staff through the ticket system.
